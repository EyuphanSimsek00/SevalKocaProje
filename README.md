# 👗 Seval Koca - Kadın Giyim E-Ticaret Arayüzü

Modern, estetik ve akıcı bir kullanıcı deneyimi sunmak amacıyla tasarlanmış kadın giyim mağazası web projesi. 

## ✨ Öne Çıkan Özellikler
* **Mobil Uyumluluk (Responsive):** Tüm cihazlarda kusursuz görünüm.
* **Dinamik UI:** Akıcı geçişler ve modern tasarım dili.
* **Hızlı Yükleme:** Optimize edilmiş bileşen yapısı.

## 💻 Kullanılan Teknolojiler
* Frontend: React 
* Dil: TypeScript
* Stil: Tailwind CSS / SCSS 

## 🚀 Kurulum ve Çalıştırma
Projeyi yerel ortamınızda çalıştırmak için:
1. Depoyu klonlayın: `git clone https://github.com/EyuphanSimsek00/SevalKocaProje.git`
2. Bağımlılıkları yükleyin: `npm install`
3. Geliştirme sunucusunu başlatın: `npm run dev`
4. Tarayıcınızda `http://localhost:3000` adresine gidin.
5. Tarayıcınızda `http://localhost:3000/admin` adresine giderek admin paneline ulaşabilirsiniz.
